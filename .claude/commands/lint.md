# Lint and Format

Run code quality checks and auto-formatting.

## Instructions

1. Run ruff format to auto-format code:
```bash
ruff format .
```

2. Run ruff check for linting:
```bash
ruff check .
```

3. If there are auto-fixable issues:
```bash
ruff check --fix .
```

Report any issues that cannot be auto-fixed and suggest manual corrections.
